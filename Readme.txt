All barcode decoding credit goes to the http://zbar.sourceforge.net/

Pre-steps:
    1. Download and install the zbar on PC.
    (version zbar-0.10-setup.exe was used in this project, install that to your PC)
    2. Add the "bin folder" to system path. It looks something like "C:\Program Files (x86)\ZBar\bin"

1. put all barcodes images under folder barcodes
2. run BarcodeReader.bat
3. wait for the middle.txt to generate
3. press enter to continue to produce the final result.txt, containing all the scaned barcodes
